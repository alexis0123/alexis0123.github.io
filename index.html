About Me

I started with Python, making small scripts to make my tasks easier. One was for a school assignment about income tax, where I made a program so I didn’t have to compute everything by hand. After that, I tried a little bit of web programming using Flask and really enjoyed it, but because of our slow internet at home, I decided to learn Android development first since I enjoy making apps for myself. Now I’m learning Kotlin and building Android apps, and I want to be a backend developer too.

Projects

Tax Calculator

A simple Python program that calculates income tax based on Philippine tax brackets. It supports different deduction types like Standard, Itemized, and OSD (Optional Standard Deduction). Built to automate a tedious school assignment where multiple tax scenarios had to be solved manually.

Finance Tracker

I made a Personal Financial Tracker app on my phone using Pydroid 3 and Kivy. It helps me track my daily expenses, cashflow, and savings. The app saves everything in an Excel file with a CSV backup. All entries are saved in one sheet with the date, category, and amount. I use it to see how I spend and save money.
